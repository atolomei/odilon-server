

<h1>Contributing to Odilon</h1> 

Thanks for your interest in contributing! Odilon is an open-source, friendly project focused on learning and growth.

We appreciate all kinds of contributions — code, ideas, translations or documentation. Contributing helps you level up while making the project better for everyone.

<h2>Odilon client SDKs</h2>
Currently we need to write client SDKs in different languages.
Odilon has a SDK in Java and the next step is to allow applications on other programming languages to use the server via a software development kit, like we have in Java.

<h2>Talk to us</h2>
If you want to work on something or have questions / complaints start a github [discussion](https://github.com/atolomei/odilon-server/discussions).
